# cpp_math_quiz_engine 🔢

A C++ Math Quiz game built to practice logic, structs, and input validation. Features 4 difficulty levels and custom randomization.  Focusing on structural data and error handling.

## 🚀 What this project does
* **Random Questions:** The game creates different math problems every time.
* **4 Difficulty Levels:** You can choose from Easy, Mid, Hard, or Mix (which uses all levels).
* **Smart Data:** I used `Structs` and `Enums` to keep track and updates the scores and the game state.
* **Fun Colors:** The console screen changes color (Green for correct, Red for wrong) to give you feedback.
* **No Crashes:** I added "Input Validation" so the game doesn't break if you accidentally type a letter instead of a number.

## 🛠️ What did I gain making this project
* How to use **Loops** and **Switch-Case** for game logic.
* Managing data with Nested Structs (Organizing questions inside a quiz).
* Handling **User Input** safely (no more infinite loops!).
* Using the **Random** library in C++.
* Preventing program crashes by implementing Buffer Clearing using cin.ignore() and cin.clear().
